# @parse5/htmlparser2-tree-adapter

## Usage

```js
const parse5 = require('parse5');
const htmlparser2Adapter = require('@parse5/htmlparser2-tree-adapter');

const document = parse5.parse('<div></div>', { treeAdapter: htmlparser2Adapter });
```
