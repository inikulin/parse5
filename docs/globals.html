treeAdapters

treeAdapters: object
  • Defined in index.d.ts:749

Provides built-in tree adapters that can be used for parsing and serialization.

example

const parse5 = require('parse5');

// Uses the default tree adapter for parsing.
const document = parse5.parse('<div></div>', {
    treeAdapter: parse5.treeAdapters.default
});

// Uses the htmlparser2 tree adapter with the SerializerStream.
const serializer = new parse5.SerializerStream(node, {
    treeAdapter: parse5.treeAdapters.htmlparser2
});

Type declaration

  • default: TreeAdapter

    Default tree format for parse5.

  • htmlparser2: TreeAdapter

    Quite popular htmlparser2 tree format (e.g. used by cheerio and jsdom).

serialize

  • serialize(node: Node, options?: SerializerOptions): string
    • Defined in index.d.ts:830

    Serializes an AST node to an HTML string.

    example
    
const parse5 = require('parse5');

const document = parse5.parse('<!DOCTYPE html><html><head></head><body>Hi there!</body></html>');

// Serializes a document.
const html = parse5.serialize(document);

// Serializes the <html> element content.
const str = parse5.serialize(document.childNodes[1]);

console.log(str); //> '<head></head><body>Hi there!</body>'

    Parameters

    • node: Node

      Node to serialize.

    • Optional options: SerializerOptions

      Serialization options.
